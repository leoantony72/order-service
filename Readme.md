# Order-Service Setup

&nbsp;

⭕Env File

```
PORT=4100
DB_USER=USERNAME
DB_PASSWORD=PASSWORD
DB_HOST=postgresdb
DB_PORT=5432
DB_DATABASE=e_commerce
```

&nbsp;

⭕Docker

```
docker-compose up --build //in the root dir
```

That's it
